# first_blog
刚学完springboot，做的一个很简单的SpringBoot+Vue.js前后端分离登录注册功能

# 博客登录注册


#### 简单介绍

该项目是一个简单易懂的前后端分离的登录注册项目，连接并和数据库进行校验。适合一些刚学完springboot和vue.js的伙伴，该项目能让你快速入门前后端分离项目的开发。



#### 技术栈

SpringBoot,Mybatis,Mysql,swagger,Vue.js+ElementUi
